# 全局规则模板

这个文件是公开模板版本，请将下面内容替换成你自己的全局 Codex 规则。

## 建议保留的结构

你可以按下面的结构编写：

### 工作环境

- 约定 Codex 默认在哪些目录内工作
- 约定是否优先使用项目内环境
- 约定何时需要申请授权

### 语言规范

- 约定默认输出语言
- 约定哪些技术标识保留原文
- 约定文档、标题、注释、自动化说明使用什么语言风格

### 交付检查

- 约定输出前需要检查什么
- 约定哪些常见错误要避免

## 示例占位

- 默认采用沙盒友好的方式工作
- 优先使用项目已有工具和项目内环境
- 在修改本地关键配置前先向用户确认

请根据你的团队或个人使用习惯，自行替换上面的占位内容。
